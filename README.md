# predictors_for_CAID4
We provide **three predictors** for CAID4 Challenge. Each package is distributed as a standalone installable archive with usage instructions and example input inside.

---

## 1. BindSegPred

 BindSegPred provides binding predictions for IDRs using a two-stage cascade model, encompassing both region-level and residue-level predictions.
