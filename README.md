# predictors_for_CAID4
We provide **three predictors** for protein sequence analysis. Each package is distributed as a standalone installable archive with model weights, example input, and usage instructions inside.

---

## 1. BindSegPred

Per-residue **binding site** prediction using a two-stage cascade model (anchor features, ProstT5 embeddings, and structural/physicochemical features).

## 2. disoSegPred

Per-residue **intrinsically disordered region (IDR)** prediction from sequence (ESM2 embeddings and disorder-related features).

## 3. BindPred

Per-residue **binding site** prediction for **long sequences**, using a single-stage Seq2Seq Transformer with ESM2 embeddings and structural features.
